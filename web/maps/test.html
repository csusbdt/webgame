Overview

This page shows how a map can be defined directly in HTML. Under this organization scheme, each map is defined directly in its own HTML file.

The page illustrates how movement in a table-based grid can be accomplished, and it shows a possible way to initiate interactions with NPCs and other triggers built into the grid.

This page responds to key press of awsd. Pressing 'w' makes the pc move up in the grid. The other key presses need to be completed.

For documentation of the jQuery functions used to solve this problem, see Traversing.
